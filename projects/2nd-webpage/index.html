This site is to display my present knowledge gathered to format/compile a very basic webpage. Further advancements will be added as and when I master them!

I'm not expecting this page to win design awards! All I'm attempting is to encompass some of the training I have developed over the last few months. Along with showing what the edition of basic CSS can do to the design of a web page.

As with this box, it's added by typing into the CSS sheet overflow:auto;. If enough wording is added, then you would have to scroll down. It has its usage, news scrolling etc. Just as I'm attempting here to rabbit on, to prove the point!

I've tried my best to keep the page as semantically correct as I know how to this stage. But, probably reading back in time I'll find a lot wrong with the mark-up! And the wireframe (layout) obviously needs some work ... but, I could spend forever here and do nothing else!

This is an example of CSS the styling for this box:

width: 350px;
