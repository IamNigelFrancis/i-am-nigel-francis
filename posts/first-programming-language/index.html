FIRST PROGRAMMING LANGUAGE

Nigel Francis

January 17th 2015

Once the cats out of the bag and you get your claws into coding, you’ll think, what now? A similiar course perhaps, which programming language first, one language or study more perhaps? The questions WILL be endless.

HTML5 (Hypertext Mark Up Language) and CSS3 (Cascading Style Sheets) are what any developer/programmer or web designer should have in their armoury, but these are not coding languages per sa. HTML (structure for a page) and CSS (visual, layout and aural) are two of the core technologies for building web pages. More on them in a later post.

Every fledging developer will ask this question and even seasoned developers still question their first choice of language. But, you could spend a lifetime reading views on WHICH LANGUAGE should I choose and nothing else. It’s an infinite subject and eventually ‘You Won’t See The Woods For The Trees‘ and it’ll drive you barmy!

Firstly, choose a path: Front-End or Back-End. You could one day practice both and become a Full-Stack-Ninja!

What do these mean, you’ll soon find out, but the crutch is: Front-End: the fancy pages that you see and interact with, the Back-End: what drives it all (frameworks/servers etc... )

The one language of note for front-end is Javascript, as it’s already built into your browser. It immediately adds interactivity to your code (on a web page) with little learning. And now, with the rise of Meteor, Javascript can be used front-end and back-end. How cool is that!

If you’re looking at a programming path, then Python is undoubtedly the one. It’s the easiest language to learn and most fun, with most Uni’s using it as the bedrock for their courses. You really can’t ask for better than that!

There’s never a correct answer for your FIRST LANGUAGE, but following the below infographic and video won’t do you any harm. Which ever language you initially follow, the next will be 50% easier to learn... so I'm told!

If you’ve considered a Microsoft-technology-path, then C# is a must, with it been well integrated into the very so powerful .NET framework. This is also becoming a serious contender for the corporate market place and plenty of jobs abound.

Or perhaps, try a few languages out first, but once settled on one, stick to it. Otherwise, you’ll just get too overwhelmed by them all and give up! 90% of people starting out on the self-taught-route, give up. It’s not easy, but it’s also not impossible, endeavor and be that 10% that SUCCEED.

As you progress, keep one thing in mind,programming languages are tools. Choose your tools after you know your task, not before and as a developer you’re always going to be at school!
